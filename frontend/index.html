Upload Arsip

Bulk Upload PDF

Petunjuk: Nama file harus sesuai format yyyymmdd_{koderegistrasi}_{kodelayanan}.pdf.
Contoh: 20250808_653_D.pdf

Struktur Folder: File akan disimpan dalam struktur folder hierarkis: arsip/yyyy/yyyymm/yyyymmdd/
Contoh: arsip/2025/202508/20250808/20250808_653_D.pdf

Fitur Convert: Anda dapat mengunggah file campuran (JPG, PNG, PDF) untuk dikonversi menjadi satu PDF dengan nama custom.

Convert & Upload File Campuran

Pilih file gambar (JPG, PNG) dan/atau PDF yang akan digabung menjadi satu file PDF
Format: yyyymmdd_nomorregistrasi_kodelayanan (tanpa .pdf)
File akan disimpan di: arsip/yyyy/yyyymm/yyyymmdd/namafile.pdf
